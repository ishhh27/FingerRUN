# 🚀 fingerRUN
A gesture-controlled 3D endless runner game built using Three.js and real-time hand tracking.
🌌 Run through a glowing galaxy world filled with neon visuals, dynamic effects, powerups, immersive music, and smooth arcade gameplay controlled entirely by gestures.
# ✨ Features
* 👋 Real-time hand gesture controls
* 🎮 Endless runner gameplay
* 🌌 Galaxy-themed environment
* ☄️ Meteors, glowing visuals, and neon effects
* 🧲 Magnet powerup system
* 🛡️ Shield protection system
* 🎵 Background music integration
* 🌈 Dynamic coin color progression
* ⚡ Smooth lane switching and jump mechanics
* 📷 Webcam-based interaction
* 🚀 Responsive arcade-style UI
---
# 🛠️ Tech Stack
* Three.js
* JavaScript
* HTML5
* CSS3
* MediaPipe / Hand Tracking
* WebGL

# 🎯 How to Play
| Gesture         | Action     |
| --------------- | ---------- |
| Move hand left  | Move left  |
| Move hand right | Move right |
| Raise hand      | Jump       |
Collect coins, avoid obstacles, and survive as long as possible while speeding through the galaxy.
